# PixelPlanetSurvival

Jeu de survie pixel art sur une planete, construit en TypeScript avec Vite et Express.

## Scripts

```bash
npm install
npm run dev
npm run build
npm start
```

- `npm run dev` lance Express avec Vite en middleware.
- `npm run build` compile le client Vite et le serveur TypeScript.
- `npm start` sert le build de production.
