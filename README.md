# Sphere Shoot

Projet TypeScript + Vite + Express repris depuis `C:\tmp\sphere_navigation.html`.

## Scripts

```bash
npm install
npm run dev
npm run build
npm start
```

- `npm run dev` lance Express avec Vite en middleware.
- `npm run build` compile le client Vite et le serveur TypeScript.
- `npm start` sert le build de production.
